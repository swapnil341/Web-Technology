 - complex DOM manipulations

SPA (Single Page Application)
 - Ex Instagra, Facebook

Virtual DOM
 - React creates a copy of real DOM its called virtual DOM.
 - When data changes Ract updates virtual DOM.
 - Updates the only changed parts in real DOM.
 - Adds high performance.
